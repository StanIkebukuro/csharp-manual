# switch文
当面はif文でよい。詳しくは以下のサイトを参照のこと。
http://ufcpp.net/study/csharp/st_branch.html#switch

以下、switch文の書き方。

```cs
switch (変数) {
//条件1
case 条件1:
     //変数が条件1の時の処理
     break;  //C#では必要
     case 条件2;
     //変数が条件2の時の処理
     break;
     //必要なだけ、caseは書ける
     //これまでの条件のいずれにもしないとき
     default:
     //その他の処理
     break;
```

## if 文との違い
- 条件判断に使うのは、特定の一つの変数
- 条件が多い時に見やすくなる(ifで分岐させるのは, 一般的に三つ)
- switch文の利用例
　- 状態　移の分岐(タイトル画面、ゲーム画面、ゲームオーバー画面、クリア画面などを分岐させるとき)
  - HHTPなどのエラーに対して処理を実　する時
